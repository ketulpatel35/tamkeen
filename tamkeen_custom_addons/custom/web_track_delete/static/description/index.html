Data Exporting Tracking

Control the export authority and track it as well.

Export Configuration

By default, users cannot export data from the system. Therefore, to do that, the admin has to grant the export permissions to the allowed users:

Grant the export permissions directly from the user profile screen.

To grant the export permission,the admin can:

  1. Go to Settings --> Users.
  2. Search for the user you want to grant him/her the permissions.
  3. Press on 'Can Export' checkbox.

Grant the export permissions to a bulk of users from the group screen.

  1. Go to Settings --> Groups.
  2. Search for 'Can Export' group and click add an item.
  3. Select for the users you want to grant them the permissions.

Export Tracking

The module can track all the export transactions through the system, where it can track the user name, export date and the exported data references.

To view the export logs, the Admin should:

  1. Go to Settings --> Export Data Tracking --> Export Logs.
  2. Interpret the information listed in the line: Responsible, time of export, the exported object, exported fields and the exported IDs.

To check the exported data, the admin can:

  1. Go to the object list view, activate the debug mode and edit the tree to add the id in the list definition () .
  2. Select/Browse the record id of the exported data now.
  3. Export the records ids of the exported data fields as it was in the export log record..
